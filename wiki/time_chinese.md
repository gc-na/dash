# [中文] Debian Almquist Shell (dash) time 使用等价: 测量命令执行时间

## 概述
`time` 命令用于测量指定命令的执行时间。它会报告命令的真实时间、用户时间和系统时间，帮助用户分析程序的性能。

## 用法
基本语法如下：
```
time [options] [arguments]
```

## 常用选项
- `-p`：以 POSIX 格式输出时间。
- `-o FILE`：将输出结果写入指定文件。
- `-a`：将输出附加到文件中，而不是覆盖。

## 常见示例
1. 测量一个简单命令的执行时间：
   ```sh
   time ls
   ```

2. 使用 `-p` 选项以 POSIX 格式输出：
   ```sh
   time -p sleep 2
   ```

3. 将输出结果写入文件：
   ```sh
   time -o output.txt sleep 3
   ```

4. 将输出附加到已有文件中：
   ```sh
   time -ao output.txt sleep 4
   ```

## 提示
- 在执行较长时间的命令时，使用 `time` 可以帮助你了解性能瓶颈。
- 如果你需要多次测量同一命令的执行时间，可以将输出结果写入文件，以便后续分析。

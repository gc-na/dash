# [中文] Debian Almquist Shell (dash) pidstat 使用方法: 监控进程统计信息

## 概述
`pidstat` 命令用于监控系统中各个进程的统计信息，包括 CPU 使用率、内存使用情况等。它可以帮助用户分析进程的性能表现。

## 使用方法
基本语法如下：
```bash
pidstat [options] [arguments]
```

## 常用选项
- `-h`：显示帮助信息。
- `-r`：显示内存使用情况。
- `-u`：显示 CPU 使用情况。
- `-p <pid>`：指定要监控的进程 ID。
- `-t`：显示线程统计信息。

## 常见示例
1. 显示所有进程的 CPU 使用情况：
   ```bash
   pidstat -u
   ```

2. 显示特定进程（例如 PID 为 1234）的内存使用情况：
   ```bash
   pidstat -r -p 1234
   ```

3. 每 2 秒更新一次所有进程的 CPU 和内存使用情况：
   ```bash
   pidstat -u -r 2
   ```

4. 显示所有线程的统计信息：
   ```bash
   pidstat -t
   ```

## 小贴士
- 使用 `-h` 选项可以快速查看 `pidstat` 的所有可用选项，帮助你更好地理解命令的功能。
- 在监控高负载系统时，适当调整更新频率可以减少系统负担。
- 结合其他命令（如 `grep`）使用，可以更方便地筛选出特定进程的信息。

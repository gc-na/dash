# [ديبيان] Debian Almquist Shell (dash) at: [جدولة المهام]

## Overview
أمر `at` يُستخدم في نظام التشغيل لجدولة تنفيذ الأوامر أو السكربتات في وقت محدد في المستقبل. يمكن للمستخدمين تحديد وقت معين لتنفيذ الأوامر، مما يسهل إدارة المهام التلقائية.

## Usage
التركيب الأساسي للأمر هو:

```bash
at [options] [arguments]
```

## Common Options
- `-f FILE`: قراءة الأوامر من الملف المحدد بدلاً من إدخالها مباشرة.
- `-m`: إرسال بريد إلكتروني للمستخدم عند انتهاء تنفيذ الأمر.
- `-q QUEUE`: تحديد قائمة الانتظار التي سيتم جدولة الأمر فيها.
- `-v`: عرض معلومات إضافية حول الأمر المجدول.

## Common Examples
- جدولة أمر بسيط للتنفيذ في وقت محدد:
```bash
echo "echo 'Hello, World!'" | at 10:00
```

- جدولة أمر لتنفيذ سكربت:
```bash
at 14:30 -f /path/to/script.sh
```

- جدولة أمر مع إرسال بريد إلكتروني عند الانتهاء:
```bash
echo "backup.sh" | at -m 02:00
```

- جدولة أمر ليتم تنفيذه في وقت لاحق من نفس اليوم:
```bash
at now + 1 hour
```

## Tips
- تأكد من أن الخدمة `atd` تعمل على النظام لديك لتتمكن من جدولة الأوامر.
- استخدم الأمر `atq` لعرض قائمة الأوامر المجدولة.
- يمكنك استخدام الأمر `atrm` لإلغاء أمر مجدول إذا لزم الأمر.

# [डेबियन] Debian Almquist Shell (dash) wget उपयोग: फ़ाइलें डाउनलोड करना

## Overview
`wget` एक शक्तिशाली कमांड-लाइन उपकरण है जिसका उपयोग वेब से फ़ाइलें डाउनलोड करने के लिए किया जाता है। यह HTTP, HTTPS और FTP प्रोटोकॉल का समर्थन करता है और बैकग्राउंड में चलने की क्षमता रखता है, जिससे उपयोगकर्ता अन्य कार्य कर सकते हैं जबकि डाउनलोड जारी रहता है।

## Usage
`wget` कमांड का मूल सिंटैक्स निम्नलिखित है:

```bash
wget [options] [arguments]
```

## Common Options
- `-O <file>`: डाउनलोड की गई फ़ाइल का नाम निर्दिष्ट करें।
- `-c`: यदि डाउनलोड अधूरा रह गया है, तो उसे फिर से जारी करें।
- `-q`: चुपचाप डाउनलोड करें, कोई आउटपुट नहीं दिखाएँ।
- `--limit-rate=<rate>`: डाउनलोड की गति को सीमित करें।
- `-r`: रीकर्सिव डाउनलोड, जिससे संपूर्ण वेबसाइट डाउनलोड की जा सकती है।

## Common Examples
1. एक साधारण फ़ाइल डाउनलोड करना:
   ```bash
   wget http://example.com/file.zip
   ```

2. फ़ाइल का नाम बदलकर डाउनलोड करना:
   ```bash
   wget -O newfile.zip http://example.com/file.zip
   ```

3. अधूरा डाउनलोड फिर से जारी करना:
   ```bash
   wget -c http://example.com/file.zip
   ```

4. रीकर्सिव डाउनलोड करना:
   ```bash
   wget -r http://example.com/directory/
   ```

5. डाउनलोड की गति सीमित करना:
   ```bash
   wget --limit-rate=200k http://example.com/file.zip
   ```

## Tips
- हमेशा `-c` विकल्प का उपयोग करें यदि आप बड़े फ़ाइलों को डाउनलोड कर रहे हैं, ताकि आप अधूरे डाउनलोड को फिर से शुरू कर सकें।
- यदि आप कई फ़ाइलें डाउनलोड कर रहे हैं, तो एक स्क्रिप्ट बनाना सहायक हो सकता है।
- `-q` विकल्प का उपयोग करें जब आप केवल आवश्यक जानकारी देखना चाहते हैं, जिससे आउटपुट साफ रहता है।

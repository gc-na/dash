# [سیستم‌عامل] Debian Almquist Shell (dash) bunzip2 استفاده: فشرده‌سازی فایل‌ها

## Overview
دستور `bunzip2` برای استخراج فایل‌های فشرده شده با فرمت Bzip2 استفاده می‌شود. این دستور به شما امکان می‌دهد تا فایل‌های .bz2 را به حالت اولیه خود برگردانید.

## Usage
سینتکس پایه دستور `bunzip2` به صورت زیر است:

```bash
bunzip2 [options] [arguments]
```

## Common Options
- `-k` : فایل فشرده شده را حفظ کرده و فقط نسخه استخراج شده را ایجاد می‌کند.
- `-f` : در صورت وجود فایل‌های با نام مشابه، آن‌ها را بدون پرسش جایگزین می‌کند.
- `-v` : اطلاعات بیشتری در مورد فرآیند استخراج نمایش می‌دهد.

## Common Examples
اینجا چند مثال عملی از استفاده از `bunzip2` آورده شده است:

1. استخراج یک فایل Bzip2:
   ```bash
   bunzip2 example.bz2
   ```

2. استخراج و حفظ فایل فشرده شده:
   ```bash
   bunzip2 -k example.bz2
   ```

3. استخراج چندین فایل Bzip2 به طور همزمان:
   ```bash
   bunzip2 file1.bz2 file2.bz2
   ```

4. استفاده از گزینه جایگزینی اجباری:
   ```bash
   bunzip2 -f existing_file.bz2
   ```

## Tips
- همیشه قبل از استخراج، از فایل‌های مهم خود نسخه پشتیبان تهیه کنید.
- اگر می‌خواهید از فایل فشرده شده استفاده کنید، گزینه `-k` را فراموش نکنید تا نسخه اصلی را حفظ کنید.
- برای مشاهده اطلاعات بیشتر در مورد فرآیند استخراج، از گزینه `-v` استفاده کنید.

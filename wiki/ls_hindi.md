# [डेबियन] Debian Almquist Shell (dash) ls उपयोग: फ़ाइलों और निर्देशिकाओं की सूची बनाना

## Overview
`ls` कमांड का उपयोग फ़ाइलों और निर्देशिकाओं की सूची देखने के लिए किया जाता है। यह कमांड वर्तमान कार्यशील निर्देशिका में मौजूद सभी फ़ाइलों और सब-निर्देशिकाओं को प्रदर्शित करता है।

## Usage
`ls` कमांड का मूल सिंटैक्स इस प्रकार है:

```bash
ls [options] [arguments]
```

## Common Options
- `-l`: विस्तृत सूची प्रारूप में फ़ाइलों की जानकारी दिखाता है।
- `-a`: छिपी हुई फ़ाइलों को भी दिखाता है (जिनका नाम "." से शुरू होता है)।
- `-h`: फ़ाइल आकार को मानव-पठनीय प्रारूप में दिखाता है (जैसे KB, MB)।
- `-R`: सभी उपनिर्देशिकाओं के साथ फ़ाइलों की सूची दिखाता है।
- `-t`: फ़ाइलों को संशोधन समय के अनुसार क्रमबद्ध करता है।

## Common Examples
यहाँ कुछ सामान्य उदाहरण दिए गए हैं:

1. वर्तमान निर्देशिका में सभी फ़ाइलों और निर्देशिकाओं की सूची दिखाने के लिए:
   ```bash
   ls
   ```

2. विस्तृत सूची प्रारूप में फ़ाइलों की जानकारी देखने के लिए:
   ```bash
   ls -l
   ```

3. छिपी हुई फ़ाइलों के साथ सूची देखने के लिए:
   ```bash
   ls -a
   ```

4. फ़ाइल आकार को मानव-पठनीय प्रारूप में दिखाने के लिए:
   ```bash
   ls -lh
   ```

5. सभी उपनिर्देशिकाओं के साथ फ़ाइलों की सूची देखने के लिए:
   ```bash
   ls -R
   ```

## Tips
- `ls` कमांड का उपयोग करते समय, आप विभिन्न विकल्पों को संयोजित कर सकते हैं, जैसे `ls -la` जो छिपी हुई फ़ाइलों के साथ विस्तृत सूची दिखाएगा।
- फ़ाइलों को क्रमबद्ध करने के लिए `-t` विकल्प का उपयोग करें, ताकि आप सबसे हाल में संशोधित फ़ाइलें पहले देख सकें।
- यदि आप किसी विशेष निर्देशिका की सूची देखना चाहते हैं, तो उस निर्देशिका का पथ प्रदान करें, जैसे `ls /path/to/directory`।

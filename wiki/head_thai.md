# [ภาษาไทย] Debian Almquist Shell (dash) head การใช้งาน: แสดงบรรทัดแรกของไฟล์

## Overview
คำสั่ง `head` ใช้เพื่อแสดงบรรทัดแรก ๆ ของไฟล์ โดยปกติจะใช้เพื่อดูข้อมูลเบื้องต้นของไฟล์ที่มีขนาดใหญ่หรือเพื่อให้เห็นข้อมูลสำคัญในไฟล์นั้น ๆ

## Usage
รูปแบบพื้นฐานของคำสั่ง `head` คือ:

```
head [options] [arguments]
```

## Common Options
- `-n <number>`: กำหนดจำนวนบรรทัดที่ต้องการแสดง (ค่าเริ่มต้นคือ 10)
- `-q`: ไม่แสดงชื่อไฟล์ก่อนแสดงผล
- `-v`: แสดงชื่อไฟล์ก่อนแสดงผล (แม้ว่าจะมีไฟล์เดียว)

## Common Examples
ตัวอย่างการใช้งานคำสั่ง `head`:

1. แสดง 10 บรรทัดแรกของไฟล์ `example.txt`:
   ```sh
   head example.txt
   ```

2. แสดง 5 บรรทัดแรกของไฟล์ `example.txt`:
   ```sh
   head -n 5 example.txt
   ```

3. แสดง 3 บรรทัดแรกของไฟล์ `example.txt` โดยไม่แสดงชื่อไฟล์:
   ```sh
   head -n 3 -q example.txt
   ```

4. แสดงบรรทัดแรกของหลายไฟล์:
   ```sh
   head file1.txt file2.txt
   ```

## Tips
- ใช้ `head` ร่วมกับคำสั่งอื่น ๆ เช่น `grep` เพื่อกรองข้อมูลที่คุณต้องการดูได้ง่ายขึ้น
- หากคุณต้องการดูบรรทัดสุดท้ายของไฟล์ ให้พิจารณาใช้คำสั่ง `tail` แทน
- คำสั่ง `head` มีประโยชน์มากในการตรวจสอบไฟล์บันทึก (log files) เพื่อดูข้อมูลล่าสุดอย่างรวดเร็ว

# [리눅스] Debian Almquist Shell (dash) ps 사용법: 프로세스 상태 보기

## Overview
`ps` 명령어는 현재 시스템에서 실행 중인 프로세스의 상태를 보여줍니다. 이 명령어를 사용하면 프로세스의 PID(프로세스 ID), 상태, CPU 사용량 등 다양한 정보를 확인할 수 있습니다.

## Usage
기본 구문은 다음과 같습니다:
```
ps [options] [arguments]
```

## Common Options
- `-e`: 모든 프로세스를 표시합니다.
- `-f`: 전체 포맷으로 프로세스 정보를 보여줍니다.
- `-u [user]`: 특정 사용자가 실행한 프로세스를 표시합니다.
- `-p [pid]`: 특정 PID를 가진 프로세스 정보를 표시합니다.
- `aux`: 모든 프로세스의 상세 정보를 보여주는 조합 옵션입니다.

## Common Examples
- 모든 프로세스 보기:
  ```bash
  ps -e
  ```

- 특정 사용자 프로세스 보기:
  ```bash
  ps -u username
  ```

- 특정 PID의 프로세스 정보 보기:
  ```bash
  ps -p 1234
  ```

- 전체 포맷으로 모든 프로세스 보기:
  ```bash
  ps -ef
  ```

- 모든 프로세스의 상세 정보 보기:
  ```bash
  ps aux
  ```

## Tips
- `ps` 명령어는 `grep`과 함께 사용하여 특정 프로세스를 쉽게 찾을 수 있습니다. 예를 들어, `ps aux | grep ssh`를 사용하면 SSH 관련 프로세스를 찾을 수 있습니다.
- `ps` 명령어는 시스템의 현재 상태를 반영하므로, 주기적으로 확인하여 시스템 리소스 사용을 모니터링하는 것이 좋습니다.
